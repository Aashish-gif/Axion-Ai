## 🎯 About

Axion AI is a comprehensive YouTube analytics platform that leverages artificial intelligence to analyze video comments, generate insights, and help creators optimize their content strategy.

### 💡 Value Proposition

- **⚡ Save Time**: Analyze 1000+ comments in 30 seconds
- **🎯 Data-Driven**: Make content decisions based on real audience data
- **🌍 Global**: Support for 23+ Indian languages
- **📈 Growth**: Increase engagement with AI-powered insights

## ✨ Features

### 🔍 Sentiment Analysis
- Real-time comment analysis with 87% accuracy
- Multi-language support for 23+ Indian languages
- Trend tracking and sentiment changes over time

### 💬 Smart Replies
- Contextual AI-generated responses
- Personalized to your channel's voice
- Batch processing for hundreds of comments

### 💡 Content Ideas Generator
- Audience request identification
- Trending topic discovery
- Performance prediction for new content

### 📊 Analytics Dashboard
- Real-time engagement metrics
- Historical trend analysis
- Exportable PDF reports

## 🛠️ Tech Stack

### Frontend
- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript 5.0+
- **Styling**: Tailwind CSS 3.4+
- **State Management**: React Context + Zustand

### Backend
- **Runtime**: Node.js 18+
- **Framework**: Next.js API Routes
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: JWT + bcrypt

### AI/ML
- **Primary**: OpenAI GPT-4 for content generation
- **Custom Models**: Sentiment analysis NLP models
- **Languages**: Multi-language NLP pipelines

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- MongoDB 6.0+
- OpenAI API key
- YouTube Data API v3 key

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/axion-ai.git
cd axion-ai

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Start MongoDB
mongod

# Run the development server
npm run dev
```

### Environment Variables

```env
# Database
MONGODB_URI=mongodb://localhost:27017/axion-ai

# Authentication
JWT_SECRET=your-super-secret-jwt-key
NEXTAUTH_SECRET=your-nextauth-secret

# APIs
OPENAI_API_KEY=your-openai-api-key
YOUTUBE_API_KEY=your-youtube-api-key

# OAuth
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
```

## 📊 API Documentation

Complete API documentation is available in `Axion-AI-API-Clean.json` - import this into Postman to test all endpoints.

### Key Endpoints

- **Authentication**: `/api/auth/*`
- **User Profile**: `/api/user/profile`
- **YouTube Integration**: `/api/youtube/*`
- **AI Features**: `/api/youtube/report`, `/api/youtube/smart-replies`
- **Settings**: `/api/dashboard/settings`
